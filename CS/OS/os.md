# OS 운영체제

운영체제란?
> 하드웨어 바로 위에 설치되는 소프트웨어 계층

커널(Kernal)은 운영체제의 핵심이 되는 프로그램을 의미한다. (메모리에 상주하는 운영체제의 실행코드 모음)
> 운영체제의 소프트웨어 어플리케이션을 하드웨어에 연결
> 하드웨어와 소프트웨어를 연결하기 위해 메모리상에서 계속 실행되는 프로세스
