### Java 예외 (Exception)
###### 2021.12.09
---

* **OutOfMemmoryError**
  * Java 힙메모리에 오브젝트를 할당하기에 공간이 불충분할 때 발생
<br>

* **RuntimeException vs Exception**
  * Exception의 경우 try / catch 로 잡아주지 않으면 컴파일 오류가 발생하지만,
    RuntimeException은 컴파일 오류가 발생하지 않는다.
    (개발자가 부주의하여 발생할 수 있는 경우에 발생하도록 만든 오류라고 생각할 수 있다.)
