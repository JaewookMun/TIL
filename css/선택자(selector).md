    border: 1px solid black;
}

* **attribute**
  *-* 특정 속성이나 속성값을 가지고 있는 HTML 요소(Element)를 선택
    (다른 선택자들과 함께 활용하여 원하는 요소를 선택할 수 있다.)

``` css
/* example */

[name="pageNo"] {
    display: none;
}

input[name="user"] {
    color: red;
}

```
