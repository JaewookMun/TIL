# JPA (Java Persistence Api)

## 데이터베이스 연결

**SQL Mapper** 
* ex) Mybatis, JdbcTemplate
* 관계형 데이터베이스의 구조에 종속적인 코드를 작성

**ORM (Object Relational Mapping) : 객체 관계 매핑**
* ex) JPA(<- Hibernate)
* 객체지향적인 프로그래밍 방식을 활용하여 코드를 작성

> JPA에서 가장 중요한 개념
> 1. 객체와 관계형 데이터베이스 연결(Mapping)
> 2. 영속성 컨텍스트
