# 도커(docker)
---

도커는 컨테이너 기반의 오픈소스 가상화 플랫폼을 의미한다.

다양한 프로그램, 실행환경을 컨테이너로 추상화하고 동일한 인터페이스를 제공하여 프로그램의 배포 및 관리를 단순하게 해준다.
