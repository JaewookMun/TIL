## 스프링 MVC 
---

ResponseEntity

JSON 형태로 데이터를 주고 받을 때 서버가 제공하는 api에는 단순히 값만 전달되지 않는다.
HTTP status code, Header 정보, 등이 존재.
ResponseEntity를 통해 이러한 정보들을 한번에 담아서 전달 가능하다.
